Computação: Serverless e Containers

A execução de código e contentores sem a necessidade de provisionar ou gerir infraestrutura subjacente.

Serviço AWS Definição Teórica e Aplicações Práticas
AWS Lambda

Executa código em resposta a eventos (triggers). Limite máximo de 15 minutos por execução. Perfeito para o "menor esforço operacional".

Casos Práticos do Simulado:
  • O Application Load Balancer suporta a integração nativa com funções Lambda e permite o roteamento de caminhos HTTP (por exemplo, /api/users para Lambda-Users, /api/orders para Lambda-Orders).
  • O Amazon Aurora Serverless v2 dimensiona a capacidade da base de dados instantaneamente e de forma fracionada (em ACUs - Aurora Capacity Units) com base na utilização real da aplicação, permitindo ...
  • O AWS Secrets Manager foi projetado especificamente para gerir, proteger e rodar segredos de forma centralizada.
  • O Amazon EventBridge é um barramento de eventos serverless que facilita a conexão de aplicações usando dados de recursos AWS, de aplicações SaaS e aplicações próprias do utilizador.
  • O AWS Step Functions é um serviço de orquestração visual serverless de fluxos de trabalho baseados em máquinas de estado.
  • A combinação API Gateway + Lambda + DynamoDB é a arquitetura de microsserviços serverless por excelência na AWS, oferecendo altíssima escalabilidade e latências de milissegundos para operações de l...
  • O Amazon Cognito fornece pools de utilizadores (User Pools) e pools de identidades (Identity Pools) serverless para gerir registos, logins, controlos de acesso e logins federados com redes sociais ...
  • O Fargate Spot fornece capacidade serverless com descontos massivos de até 70% comparado ao Fargate padrão, perfeito para tarefas tolerantes a falhas rápidas.
  • O AWS Lambda é o principal serviço de computação serverless da AWS.
  • O combo API Gateway, Lambda e DynamoDB é a arquitetura Serverless definitiva recomendada pela AWS, apresentando escalonamento instantâneo, redundância Multi-AZ ativa e custo baseado puramente no us...
