Segurança: Proteção e Defesa Web

Prevenção de intrusões, análise de ameaças e encriptação.

Serviço AWS Definição Teórica e Aplicações Práticas
AWS WAF (Web Application Firewall)

Defende contra explorações a nível da aplicação (Layer 7), como SQL Injections, Cross-Site Scripting (XSS) e bloqueia listas de IPs em endpoints HTTP.

Casos Práticos do Simulado:
  • O AWS WAF filtra requisições na camada 7 (HTTP/HTTPS) contra injeções SQL e Cross-Site Scripting (XSS).
  • O AWS WAF protege as camadas de aplicação contra SQLi e XSS, enquanto o CloudFront e o Shield Advanced defendem de forma integrada contra ataques DDoS volumétricos (camadas 3/4/7).
AWS Shield

Defesa contra ataques de negação de serviço (DDoS) nas camadas 3 e 4. O Shield Advanced oferece proteção financeira e atua ativamente durante ataques massivos.

Casos Práticos do Simulado:
  • O AWS WAF filtra requisições na camada 7 (HTTP/HTTPS) contra injeções SQL e Cross-Site Scripting (XSS).
  • O AWS WAF protege as camadas de aplicação contra SQLi e XSS, enquanto o CloudFront e o Shield Advanced defendem de forma integrada contra ataques DDoS volumétricos (camadas 3/4/7).
AWS KMS (Key Management Service)

A espinha dorsal para encriptação em repouso (at rest). Cria e gere chaves de encriptação (CMKs).

AWS Secrets Manager

Serviço projetado especificamente para gerir, proteger e RODAR AUTOMATICAMENTE segredos (chaves de API, credenciais de BD) usando funções Lambda embutidas.

Casos Práticos do Simulado:
  • O AWS Secrets Manager foi projetado especificamente para gerir, proteger e rodar segredos de forma centralizada.
AWS Systems Manager Parameter Store

Armazenamento centralizado para dados de configuração e senhas (SecureString). Não possui funcionalidade nativa de rotação automática de segredos como o Secrets Manager.

Amazon Macie

Serviço de segurança baseado em Machine Learning dedicado a identificar informações confidenciais PII/dados sensíveis expostos em buckets S3.

Teoria fundamental para o exame. Não testado diretamente nos seus simulados.
Amazon GuardDuty

Detetor de ameaças inteligente que monitoriza logs do CloudTrail, VPC Flow Logs e consultas de DNS em tempo real para encontrar comportamento anómalo.

Teoria fundamental para o exame. Não testado diretamente nos seus simulados.
